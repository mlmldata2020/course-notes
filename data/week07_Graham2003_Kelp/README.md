Variables:

* Response - shallow (upper) distributional limit of kelp Macrocystis pyrifera [m]
* OD - wave orbital displacement [m]
* BD - wave breaking depth [m]
* LTD - minimum daily tidal height [m]
* W - wind velocity [m/s], positive onshore
